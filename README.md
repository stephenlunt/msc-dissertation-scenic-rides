## Project structure

```
$ ROOT
│   # Application entrypoint
├── App.tsx
│
│   # Source code files
├── src
│   │   # Assets compiled into the application
│   ├── assets
│   │   # Reusable React components
│   ├── components
│   │   # Application data store
│   ├── data
│   │   # Application screens (pages)
│   ├── screens
│   │   # Helper functions
│   └── util
│
│   # Configuration files
├── .prettierrc.js
├── app.json
├── babel.config.js
├── package.json
└── tsconfig.json
```
